a. Quem inventou o sistema de controle de versão Git e por quê?

O Git foi desenvolvido em 2005 por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux. Com o obijetivo de se ter o controle sobre o histórioco de um arquivo de código, sendo possível ter um acompanhamento sobre todas as modificações que foram feitas.

b. Quem atualmente é o dono do Github e por quê?

Atualmente a Microsoft é a dona do GitHub, a compa foi feita porque ela é uma companhia voltada ao desenvolvedor e, ao unir forças com o GitHub, ela fortalece seu comprometimento com a liberdade do desenvolvedor, abertura e inovação.

c. Existe outra maneira, além do terminal, de trabalhar com o Github?

Exíte a opção de usar os recursos visual disponibilizados no site oficial ou/e pelos aplicativos, mas a experiência é mais limitada do que pelo terminal.
